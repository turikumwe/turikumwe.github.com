Getting started with Hyde website

Posted: Tue, 01 Feb 2011
  • happy
  • thoughts

This step by step tutorials will help you to get started with Hyde Blog. t will also guide when you will be designing your Web pages using some content management(CMS).

Contents

1. Installing PIP
2. Installing Hyde
3. Generating your default website
4. Publish Your Website

1.Installing PIP


pip is a tool for installing and managing Python packages, such as those found in the Python Package Index. Once you have successfuly installed pip then you can use

pip install SomePackage
to install Python packages. On your command line type:
$ easy_install pip

2.Installing Hyde


Installing Hide requires only a single command if PIP is installed
$ pip install hyde


This command will download and setup your Hyde static website generator engine for You.

3.Generating your Hyde website


To generate your first website a few steps are required.

3.1 The create command


To create a new website your must issue the following command:
$ hyde create

Note that this command will create the website in the working direcory! otherwise you will have to specify the directory in your commannd.

3.2 The generate command

To generate your site you’ll have to issue the following command. Note that generate will make your created website ready to be deployed.

$ hyde gen

3.3 The serve command

Starts the built in web server that also regenerates based on the request if there are changes. This command also start the Hyde built in web server.

$ hyde serve

If the out put of this command are like this, You are done with your new Hyde website. Open your browser and the the provided URL(Ex: http://localhost:8080) for our case:

23:44:05 hyde.engine Reading site configuration from [/Users/nyaruka/t/site.yaml]
23:44:06 hyde Starting webserver at [localhost]:[8080]

4. Publishing Your Website on gitHub.com

Once you have found that your site is done and running properly on your localhost, then you can upload it to the gitahub sever. Before publishing you should make change to the configuration file of your site by including the publisher, the path and the url. By default your configuration file is site.ymal in your site directory. Open it in your editor or use emacs and add the following lines at:

publisher:
        github:
            type: hyde.ext.publishers.dvcs.Git
            path: ../your_username.github.com
            url: git@github.com:your_username/your_username.github.com.git
After you done this issue the following command to pupblish your website on gitHub.com:
$ hyde publish -p github
